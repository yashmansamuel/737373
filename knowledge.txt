# Topic: Number Theory – Advanced Concepts
Definition: Study of integers, prime numbers, divisibility, modular arithmetic, and algebraic structures.
Key Concepts:
- Modular Arithmetic: a ≡ b (mod n) iff n divides (a-b)
- Prime Gaps: Differences between consecutive primes, bounded by results like Cramér’s conjecture
- Diophantine Equations: Polynomial equations with integer solutions; solvable using modular constraints and bounds
- Euler’s Totient Function: φ(n) counts integers ≤ n coprime with n; used in RSA cryptography
Advanced Techniques:
- Lifting The Exponent Lemma (LTE) for p-adic valuations
- Quadratic Reciprocity for prime analysis
- Sieve methods (Eratosthenes, Atkin) for fast prime enumeration
Keywords: number theory, primes, modular arithmetic, Diophantine, Euler totient, LTE
